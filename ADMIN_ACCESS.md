# كيفية الدخول إلى لوحة الإدارة

## الخطوات:

1. **أضف متغيرات البيئة في ملف `.env`:**

```env
ADMIN_BASIC_USER=admin
ADMIN_BASIC_PASS=your_secure_password_here
```

⚠️ **مهم:** استبدل `your_secure_password_here` بكلمة مرور قوية!

2. **شغّل المشروع:**
```bash
npm run dev
```

3. **افتح المتصفح واذهب إلى:**
```
http://localhost:3000/admin
```

4. **ستظهر لك نافذة تسجيل الدخول:**
   - **Username:** `admin` (أو القيمة التي وضعتها في `ADMIN_BASIC_USER`)
   - **Password:** كلمة المرور التي وضعتها في `ADMIN_BASIC_PASS`

5. **بعد تسجيل الدخول، ستصل إلى لوحة التحكم!**

---

## الصفحات المتاحة:

- `/admin` - لوحة التحكم الرئيسية
- `/admin/products` - إدارة المنتجات
- `/admin/products/add` - إضافة منتج جديد
- `/admin/categories` - إدارة الكاتيجوري
- `/admin/categories/add` - إضافة كاتيجوري جديدة

---

## ملاحظات أمنية:

- لا تشارك بيانات الدخول مع أي شخص
- استخدم كلمة مرور قوية
- في الإنتاج، استخدم متغيرات بيئة آمنة
