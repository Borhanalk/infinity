# دليل تسجيل دخول المسؤول

## ✅ تم إعداد النظام بنجاح!

### بيانات الدخول:
- **البريد الإلكتروني:** `admin@infinity.com`
- **كلمة المرور:** `A@123`

---

## كيفية تسجيل الدخول:

### 1. افتح المتصفح واذهب إلى:
```
http://localhost:3002/admin/login
```

### 2. أدخل بيانات الدخول:
- البريد الإلكتروني: `admin@infinity.com`
- كلمة المرور: `A@123`

### 3. اضغط على "تسجيل الدخول"

### 4. سيتم إعادة توجيهك تلقائياً إلى لوحة التحكم

---

## الحماية:

✅ **تم تطبيق:**
- تشفير كلمات المرور (bcrypt)
- JWT tokens للتحقق
- Middleware لحماية صفحات الإدارة
- Cookies آمنة لحفظ Token

✅ **المسؤول تم إنشاؤه:**
- تم حذف جميع المسؤولين السابقين
- تم إنشاء مسؤول جديد بالبيانات المحددة
- كلمة المرور مشفرة في قاعدة البيانات

---

## تسجيل الخروج:

اضغط على "تسجيل الخروج" في أعلى لوحة الإدارة، أو اذهب إلى:
```
http://localhost:3002/admin/logout
```

---

## ملاحظات:

- إذا نسيت كلمة المرور، شغّل `npm run create-admin` مرة أخرى
- Token صالح لمدة 7 أيام
- جميع صفحات `/admin/*` محمية وتتطلب تسجيل الدخول
